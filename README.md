# Regexp? Quiz

## 📖 サービス概要
正規表現の4択クイズができる学習用Webアプリです。

## 🎯 サービス作成理由
プログラミングスクールの「アウトプット会」にて正規表現について登壇しました。<br>
登壇内容を活かして「正規表現をクイズ形式で学べるアプリ」を作成しました。<br>
覚えるのを避けがちな正規表現を気軽にインプットできるようにすることが目的です。<br>
[アウトプット会資料はこちら](https://gamma.app/docs/-c7d0jk2wuw4mncx)


## 👤 想定ユーザー
- 正規表現をこれから学びたい方
- クイズ形式で繰り返し学習したい方

## 🛠️ 技術スタック
- Docker
- Ruby (Sinatra)
- Tailwind CSS
- SQLite
- Render
